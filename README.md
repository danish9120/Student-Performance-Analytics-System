# 🎓 Student Performance Analytics System

A web-based application built using Django that helps manage student records, track academic performance, calculate percentages, and display results through an interactive dashboard.

## 🚀 Features

* Add and manage student records
* Store subject-wise marks
* Calculate percentage automatically
* Display Pass/Fail status
* Search students by Roll Number
* Responsive Bootstrap dashboard
* Django Admin Panel for data management

## 🛠️ Tech Stack

* Python
* Django
* SQLite
* Bootstrap 5
* HTML/CSS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/danish9120/Student-Performance-Analytics-System.git
```

2. Navigate to the project folder

```bash
cd Student-Performance-Analytics-System
```

3. Install dependencies

```bash
pip install django
```

4. Run migrations

```bash
python manage.py migrate
```

5. Start the server

```bash
python manage.py runserver
```

6. Open in browser

```text
http://127.0.0.1:8000/
```
